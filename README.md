Passkey sr25519 Wallet Generator

Code for the iFrame to be able to generate the underlying sr25519 key required for use with Frequency Passkey Wallets

## Development Quick Setup

1. 'npm i`
2. 'npm run dev`
3. Visit http://localhost:3000
